# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Create a RESTful API using the FastAPI framework that handles basic CRUD operations for managing items.

## 📝 Tasks

### 🛠️ Build a REST API

#### Description
Implement a simple API for managing a collection of items (e.g., a todo list or products) using FastAPI.

#### Requirements
Completed API should:

- Include endpoints for GET, POST, PUT, and DELETE operations
- Use appropriate HTTP status codes
- Handle request and response data validation
- Include basic error handling
- Provide interactive API documentation via FastAPI's built-in docs
